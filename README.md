## Run Locally

Projeyi komut satırı yardımıyla klonla

```bash
  git clone https://github.com/melihcelik09/Menthor-Frontend.git
```

Proje klasörüne git

```bash
  cd Menthor-Frontend

```

Bağımlılıkları Yükle

```bash
  npm install
```

Projeyi Çalıştır

```bash
  npm run dev
```
